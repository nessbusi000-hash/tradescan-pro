# Guide Smart Money Concept (SMC)

## Introduction

Le **Smart Money Concept (SMC)** est une approche d'analyse technique qui étudie le comportement des institutions et des gros joueurs du marché ("Smart Money") pour anticiper les mouvements de prix.

## Philosophie SMC

### Qui contrôle le marché ?

Le marché est contrôlé par des acteurs institutionnels :
- **Banques d'investissement** (Goldman Sachs, JP Morgan...)
- **Fonds de pension**
- **Hedge funds**
- **Market makers**

Ces acteurs gèrent des capitaux immenses et leurs ordres créent les mouvements de prix significatifs.

### Pourquoi le SMC fonctionne ?

1. **Information asymétrique** - Les institutions ont plus d'informations
2. **Volume** - Leurs ordres bougent le marché
3. **Patterns répétitifs** - Ils utilisent des stratégies similaires
4. **Liquidité** - Ils ont besoin de contreparties pour leurs positions

## Concepts fondamentaux

### 1. Liquidité (Liquidity)

**Définition** : Zones où se concentrent les ordres en attente (stops et limits).

**Types** :
- **Equal Highs** - Liquidité au-dessus des sommets
- **Equal Lows** - Liquidité en-dessous des creux
- **Trendline liquidity** - Liquidité sur les lignes de tendance

**Utilisation** : Les institutions viennent chercher cette liquidité avant d'inverser.

### 2. Order Blocks (OB)

**Définition** : Zones d'accumulation ou distribution institutionnelle.

**Types** :
- **Bullish OB** - Dernière bougie baissière avant un pump
- **Bearish OB** - Dernière bougie haussière avant un dump

**Utilisation** : Zones de support/résistance institutionnelles.

### 3. Fair Value Gaps (FVG)

**Définition** : Déséquilibres de prix créés par des mouvements violents.

**Structure** : 3 bougies où le milieu saute une zone.

**Utilisation** : Le prix revient souvent combler ces gaps.

### 4. Break of Structure (BOS)

**Définition** : Continuation de la tendance actuelle.

**Signification** : Les institutions continuent dans la même direction.

**Utilisation** : Confirmer la tendance, ajouter à sa position.

### 5. Change of Character (CHoCH)

**Définition** : Changement de comportement du prix.

**Signification** : Potentiel retournement de tendance.

**Utilisation** : Alerte, préparer un changement de direction.

## Analyse multi-timeframes

### Principe

L'analyse doit se faire sur plusieurs timeframes pour confirmer les signaux.

### Hiérarchie

```
HTF (Higher Time Frame) - Tendance principale
    │
    ▼
ITF (Intermediate Time Frame) - Setup
    │
    ▼
LTF (Lower Time Frame) - Exécution
```

### Exemple

- **HTF (H4)** : Tendance haussière
- **ITF (H1)** : FVG haussier formé
- **LTF (M15)** : Retest du FVG = Entrée

## Setups de trading

### Setup 1: Sweep + CHoCH

**Conditions** :
1. Sweep de liquidité
2. CHoCH dans la direction opposée
3. Entrée sur retest

**Exemple** :
- Equal Lows formés
- Prix descend et prend les lows
- Rejet violent
- CHoCH haussier
- Entrée achat

### Setup 2: FVG Retest

**Conditions** :
1. FVG significatif formé
2. Prix revient dans le FVG
3. Rejet visible
4. Entrée dans la direction du FVG

**Exemple** :
- FVG haussier H1
- Prix corrige vers le FVG
- Mèche + rejet
- Entrée achat

### Setup 3: BOS + FVG

**Conditions** :
1. BOS confirmé
2. FVG dans la direction du BOS
3. Entrée sur retest du FVG

**Exemple** :
- Bullish BOS H4
- FVG haussier H1
- Retest du FVG
- Entrée achat

### Setup 4: OB + Sweep

**Conditions** :
1. Order Block identifié
2. Sweep de la liquidité au-delà de l'OB
3. Rejet dans l'OB
4. Entrée

**Exemple** :
- Bullish OB H4
- Equal Lows en-dessous
- Sweep des lows
- Rejet dans l'OB
- Entrée achat

## Gestion des risques

### Règle des 1%

Ne jamais risquer plus de 1-2% du capital par trade.

### Ratio Risk/Reward

Minimum 1:2, idéalement 1:3 ou plus.

### Position sizing

```
Position Size = (Capital × Risk%) / (Entry - Stop)
```

### Stop Loss

Toujours placer le stop au-delà d'un niveau technique :
- Au-delà du FVG
- Au-delà du sweep
- Au-delà de l'OB

## Psychologie du trading SMC

### Patience

Attendre que tous les éléments s'alignent. Ne pas forcer les trades.

### Discipline

Suivre son plan de trading. Ne pas déplacer le stop.

### Gestion des émotions

Accepter les pertes. Se concentrer sur le processus, pas sur le résultat.

## Erreurs courantes

### 1. Over-trading

Trader trop fréquemment. Attendre les setups A+.

### 2. Ignorer le contexte HTF

Un setup LTF sans confirmation HTF est faible.

### 3. Stop mal placé

Ne pas respecter les niveaux techniques.

### 4. Greed (cupidité)

Ne pas prendre ses profits. Viser l'optimum.

### 5. Revenge trading

Vouloir récupérer une perte immédiatement.

## Checklist avant trade

- [ ] Tendance HTF identifiée
- [ ] Setup SMC clair
- [ ] Liquidité identifiée
- [ ] Point d'entrée défini
- [ ] Stop loss placé
- [ ] Target défini
- [ ] Ratio R:R > 1:2
- [ ] Risque < 2% du capital
